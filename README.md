# thunderobot_zero_bios

This hosts the original backup of thunderobot zero 11gen i7+3060 bios file and the unlocked bios file with all features.
advanced unlock(i7+3060).fd ---- unlocked bios
backup.fd ---- default bios by OEM

If you want to flash the bios available on this repo or backup your own bios all the .bat scripts are available and labeled.
